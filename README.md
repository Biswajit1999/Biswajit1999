## Current Work: EXOhSPEC

Closed-loop spectrograph stabilisation combining thermal control (TEC) and adaptive optics to hold image position stable at sub-pixel precision. Solving the thermal + mechanical drift problem that limits RV precision for exoplanet detection.

**Latest run:** 33 h hybrid (TEC + AO) — dY RMS 0.152 px, 100% feedback frames within ±0.5 px.

Manuscript in preparation for RASTI.
