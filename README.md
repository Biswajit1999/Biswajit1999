## 🏗️ Current Flagship Project: **EXOhSPEC**

Closed-loop spectrograph stabilisation system combining:
- **TEC (thermoelectric cooler)** — primary slow actuator for thermal stabilization
- **Tip–tilt adaptive optics** — secondary fast actuator for image drift correction
- **Feedback control** — PID + temperature-to-OPL modelling with rolling baseline refit

| Configuration | Performance |
|---|---|
| **Hybrid (TEC + AO), 33 h closed-loop** | dY RMS ≈ 0.152 px · 100% feedback within ±0.5 px · zero AO-unload events |
| **TEC-only baseline** | dY RMS ≈ 0.316 px — quantifies AO's ~2× improvement |
| **Dominant environmental coupling** | HVAC temperature cycling (r ≈ −0.71 with dY) |

**Status:** Multi-version factorial experiment series complete (V11 hybrid, V12 TEC-only, V13 AO-only, V14 three-phase hybrid, V14.2 extended). Manuscript in preparation for RASTI. Post-run cleanup in progress.
